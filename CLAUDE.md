# Dioptra Website — Claude Code Instructions

> Blazor dual-mode (Server + WebAssembly) marketing and contact website for Dioptra Limited

## Development Guide

Read and follow all rules in `.github/copilot-instructions.md` — it is the single source of truth for project conventions, coding standards, and architecture.

## Available Commands (via Deno)

| Command             | Purpose             |
| :------------------ | :------------------ |
| `deno task build`   | Build the solution  |
| `deno task rebuild` | Clean then rebuild  |
| `deno task aspire`  | Run via .NET Aspire |
